# 招标助手 Pro - 产品需求文档

## 产品概述
专业招标信息监控与分析工具，帮助用户实时获取招标信息、智能筛选匹配项目、生成投标建议。

## 核心功能

### 1. 招标监控
- 支持平台：一汽、东风、零跑（可扩展）
- 自动抓取频率：每 30 分钟
- 推送方式：App 推送、飞书、钉钉、企业微信

### 2. 智能筛选
- 关键词匹配（区域、新能源、活动、运营等）
- 自定义筛选规则
- 智能推荐算法

### 3. 投标分析
- 竞争对手分析
- 中标概率预测
- 投标建议生成

### 4. 数据管理
- 历史招标记录
- 收藏关注
- 数据导出

## 用户界面

### 主要页面
1. 首页 - 今日最新招标
2. 筛选页 - 自定义筛选条件
3. 分析页 - 投标建议与竞争分析
4. 我的 - 个人设置、订阅管理

## 技术架构

### 前端（iOS）
- SwiftUI
- 支持 iOS 16+

### 后端
- Node.js + Express
- PostgreSQL
- Redis

### AI 服务
- OpenAI API
- 自研筛选算法

## 商业模式
- 订阅制：¥68/月，¥298/年
- 免费版：每日限 10 条推送
- 专业版：无限推送 + AI 分析
