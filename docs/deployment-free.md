# 招标助手 Pro - 零成本部署方案

## 技术栈选择（全部免费）

### 后端服务
- **平台**: Vercel (Serverless)
- **语言**: Node.js
- **费用**: ¥0 (每月 100GB 带宽，足够起步)

### 数据库
- **平台**: Supabase
- **类型**: PostgreSQL
- **费用**: ¥0 (500MB 存储，足够起步)
- **包含**: 数据库 + 认证 + 实时订阅

### 定时任务
- **方案**: GitHub Actions
- **费用**: ¥0 (每月 2000 分钟)
- **用途**: 每30分钟触发抓取

### 域名
- **方案**: Freenom (.tk / .ml / .ga)
- **费用**: ¥0 (需每年续期)
- **备选**: 使用 Vercel 自带域名
