# Google Play 上架指南

## 1. 注册开发者账号

### 步骤
1. 访问 https://play.google.com/console
2. 使用 Google 账号登录
3. 支付一次性注册费 $25（约 ¥180）
4. 填写开发者信息
5. 等待验证（通常几分钟到几小时）

## 2. 创建应用

### 步骤
1. 点击 "创建应用"
2. 选择默认语言：简体中文
3. 应用名称：招标助手 Pro
4. 选择应用类型：应用
5. 选择类别：商务
6. 点击 "创建"

## 3. 填写应用详情

### 应用信息
- **简短说明**（80字符）：专业招标信息监控工具，实时获取招标信息
- **完整说明**（4000字符）：
```
招标助手 Pro 是一款专业的招标信息监控与分析工具，帮助您实时获取招标信息、智能筛选匹配项目。

核心功能：
• 实时监控：覆盖一汽、东风、零跑等主流招标平台
• 智能筛选：根据关键词、地区、类型精准匹配
• 投标分析：AI 智能分析中标概率
• 及时推送：重要招标信息第一时间推送
• 收藏管理：关注感兴趣的招标项目

适用人群：
• 汽车行业供应商
• 招投标从业人员
• 商务拓展人员
• 企业采购人员

免费使用基础功能，订阅专业版解锁全部功能。
```

### 图形资源
- **应用图标**：512x512 PNG
- **置顶大图**：1024x500 PNG
- **截图**：
  - 手机：至少 2 张（推荐 4-8 张）
  - 尺寸：1080x1920 或 1920x1080

## 4. 上传 APK

### 步骤
1. 选择 "正式版"
2. 点击 "创建新版本"
3. 上传签名后的 APK 文件
4. 填写版本说明
5. 点击 "保存"

## 5. 设置定价和分发

### 定价
- 应用类型：免费
- 应用内商品：
  - 月度订阅：¥68
  - 年度订阅：¥298

### 分发
- 选择国家/地区：中国及其他目标市场
- 内容分级：填写问卷（通常评为"所有人"）

## 6. 内容分级

### 填写问卷
1. 点击 "内容分级"
2. 填写电子邮件地址
3. 回答分类问题（通常都选"否"）
4. 点击 "保存"
5. 点击 "应用分级"

## 7. 商店设置

### 应用类别
- 类别：商务
- 标签：招标、投标、采购、商务

### 联系方式
- 网站：https://bid-assistant-pro.vercel.app
- 电子邮件：support@bid-assistant-pro.com
- 电话：（可选）

## 8. 隐私政策

### 添加隐私政策链接
- 隐私政策：https://bid-assistant-pro.vercel.app/privacy.html
- 勾选 "应用会访问、收集、分享或保留用户数据"

## 9. 提交审核

### 最终检查清单
- [ ] 所有必填信息已填写
- [ ] APK 已上传并签名
- [ ] 截图已上传
- [ ] 隐私政策已添加
- [ ] 内容分级已完成
- [ ] 定价已设置

### 提交
1. 点击 "发布概览"
2. 检查所有项目
3. 点击 "送审"
4. 等待审核（通常 1-3 天）

## 10. 审核通过

审核通过后：
- 应用会在几小时内在 Google Play 上架
- 你会收到电子邮件通知
- 可以通过链接分享应用

## 快速链接

- Google Play Console: https://play.google.com/console
- 开发者政策: https://play.google.com/about/developer-content-policy/
- 上架检查清单: https://developer.android.com/distribute/best-practices/launch/launch-checklist
