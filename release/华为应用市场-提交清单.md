# 华为应用市场 - 快速提交清单

## 第一步：注册账号（10分钟）

1. 访问 https://developer.huawei.com/consumer/cn/appgallery/
2. 点击右上角「注册」
3. 选择「个人开发者」
4. 填写基本信息（姓名、身份证、手机号）
5. 完成实名认证（上传身份证正反面）
6. 等待审核（通常几分钟到几小时）

**注意**：华为审核较严格，可能需要补充材料

## 第二步：创建应用（2分钟）

1. 登录后点击「我的应用」
2. 点击「新建」
3. 选择「App（APK）」
4. 填写应用名称：招标助手 Pro
5. 选择分类：商务办公 > 商务工具
6. 点击「创建」

## 第四步：上传APK

1. 点击「软件包管理」
2. 点击「上传」
3. 选择APK文件：bid-assistant-pro-v1.0.0.apk
4. 等待解析完成
5. 确认版本号、包名等信息

## 第五步：上传图标和截图

### 应用图标
- **尺寸**：216x216 像素
- **格式**：PNG
- **大小**：不超过2MB
- **背景**：透明
- **建议**：使用蓝色渐变背景，白色文字"招"

### 应用截图（需要3-5张）
- **尺寸**：450x800 或 800x450
- **格式**：JPG 或 PNG
- **大小**：每张不超过2MB
- **要求**：至少3张，最多5张

**截图内容建议**：
1. 首页招标列表
2. 招标详情页
3. 筛选功能页面
4. 收藏页面
5. 个人中心

**使用截图模板**：
打开 `release/截图模板.html`，按 F12 选择设备尺寸截图

3. 等待审核结果（3-5个工作日）

## 常见问题

### Q: 审核被拒怎么办？
A: 华为审核较严格，常见原因：
- 缺少软著或免责函 → 补充材料
- 功能不完善 → 完善功能
- 隐私政策问题 → 更新隐私政策
- 截图不清晰 → 重新截图

### Q: 多久能审核通过？
A: 首次提交通常3-5个工作日，比小米慢一些

### Q: 可以更新版本吗？
A: 可以，随时上传新版本

### Q: 收费吗？
A: 华为应用市场上架完全免费
