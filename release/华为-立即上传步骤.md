1. 点击 **"新建"** 或 **"创建应用"**
2. 选择 **"App (APK)"**
3. 填写应用名称：**招标助手 Pro**
4. 选择分类：**商务办公 > 商务工具**
5. 点击 **"创建"**

### 第4步：上传APK
1. 找到 **"软件包管理"** 或 **"上传APK"**
2. 点击上传
3. 选择文件：`bid-assistant-pro-v1.0.0.apk`
4. 等待解析完成

### 第5步：上传图标
- 尺寸：**216x216 像素**
- 格式：PNG
- 可以用这个在线工具生成：https://www.svgviewer.dev/
- 打开 `release/app-icon.svg` 转换后上传

### 第6步：上传截图
- 尺寸：**450x800 像素**
- 需要 **3-5张**
- 打开 `release/华为截图模板.html`
- 按 F12 → 设备工具栏 → 添加 450x800 尺寸 → 截图

### 第7步：填写隐私政策
- 隐私政策URL：
```
https://bid-assistant-pro.vercel.app/privacy.html
```

### 第8步：上传免责函（重要）
如果没有软件著作权：
1. 打开 `release/免责函模板.txt`
2. 打印并签字
3. 拍照或扫描
4. 在 **"资质证明"** 处上传

### 第9步：提交审核
1. 检查所有信息
2. 点击 **"提交审核"**
3. 等待审核（3-5个工作日）

A: 可以用免责函代替，打印签字上传即可
