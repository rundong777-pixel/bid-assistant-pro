# 小米应用商店 - 快速提交清单

## 第一步：注册账号（5分钟）

1. 访问 https://dev.mi.com
2. 点击右上角「开发者登录」
3. 使用小米账号登录（没有就注册）
4. 点击「实名认证」
5. 选择「个人开发者」
6. 上传身份证正反面照片
7. 等待审核（通常几分钟到几小时）

## 第二步：创建应用（2分钟）

1. 登录后点击「应用发布」
2. 点击「创建应用」
3. 选择「移动应用」
4. 填写应用名称：招标助手 Pro
5. 点击「创建」

## 第四步：上传APK

1. 点击「上传APK」
2. 选择文件：bid-assistant-pro-v1.0.0.apk
3. 等待解析完成
4. 确认包名、版本号等信息

## 第五步：上传图标和截图

### 应用图标
- 尺寸：512x512 像素
- 格式：PNG
- 背景：透明或纯色
- 建议：使用蓝色渐变背景，白色文字"招"

### 应用截图（需要5-8张）
尺寸：1080x1920 或 1920x1080
格式：JPG 或 PNG

截图内容建议：
1. 首页招标列表
2. 招标详情页
3. 筛选功能页面
4. 收藏页面
5. 个人中心
6. 关于页面

**如果你没有截图，可以先跳过这一步，后续补充**

## 第六步：提交审核

1. 检查所有信息是否填写完整
2. 勾选「我已阅读并同意小米应用商店开发者协议」
3. 点击「提交审核」
4. 等待审核结果（1-3个工作日）

## 审核通过后的操作

1. 会收到短信和邮件通知
2. 应用会在小米应用商店上架
3. 获得应用详情页链接
4. 可以分享链接给用户下载

## 常见问题

### Q: 审核被拒怎么办？
A: 根据拒绝原因修改：
- 功能不完善：补充功能
- 隐私政策问题：更新隐私政策
- 截图不清晰：重新截图
- 其他问题：联系客服

### Q: 多久能审核通过？
A: 通常1-3个工作日，首次提交可能稍慢

### Q: 可以更新版本吗？
A: 可以，随时上传新版本

### Q: 收费吗？
A: 小米应用商店上架完全免费
